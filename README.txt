 Customer Service Bot --RVNP

This project is a customer service bot designed for the RVNP website. 
It provides users with an interactive chat interface to handle customer inquiries efficiently. 
The bot is built using HTML, CSS, JavaScript, and Flask for the backend.

Features

- Chat Interface
- Message Handling
- Dynamic Content
- Static Assets

Installation

Prerequisites

- Python 3.5xx
- Flask
